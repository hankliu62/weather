# weather - 天气预报

## 简介

本项目是一个查询每日天气，以及对未来几天天气进行预报的Web APP，数据仅供参考，无事常备雨伞，总不会有错。

没有伞的孩子只能努力奔跑。

## 环境依赖

- Node.js：v18 及以上版本
- pnpm.15.5 及以上版本
- 浏览器：Chrome、Firefox、Safari 等现代浏览器

## 安装步骤

1. 克隆项目代码到本地

```bash
git clone https://github.com/hankliu62/weather.git
```

2. 进入项目目录

```bash
cd weather
```

3. 安装项目依赖

```bash
pnpm install
```

4. 启动项目

```bash
pnpm run dev
```

## 使用方法

访问 `http://localhost:3000` 即可看到人生一格的首页。

在首页上可以选择需要的工具进行使用。

工具的具体使用方法和参数可以参考每个工具的文档说明。
